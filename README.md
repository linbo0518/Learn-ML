### Environment:

Python: 3.6.X

### Install:

install `numpy` `scipy` `graphviz` `matplotlib` `scikit-learning` via pip:

```bash
pip3 install numpy scipy graphviz matplotlib scikit-learning
```

clone into your folder:

```git
git clone https://github.com/linbo0518/learn_ml.git
```

### Run:

```Python
Python3 filename.py
```
