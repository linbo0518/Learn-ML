### Environment:

OS: macOS 10.13

Python: 3


### Install:

install `numpy`, `pandas`, `matplotlib`, `graphviz`, `scikit-learn` via pip:

```bash
pip3 install numpy pandas matplotlib graphviz scikit-learn
```

clone into your folder:

```bash
git clone https://github.com/linbo0518/learn_ml.git
```


### Run:

```bash
python3 filename.py
```
